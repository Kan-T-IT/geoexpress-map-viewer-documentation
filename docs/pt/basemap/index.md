# Seletor de Mapa Base

No canto inferior direito da tela, há um quadrado com um mapa dentro, que permite ao usuário selecionar o mapa base do visualizador, sobre o qual todas as camadas são exibidas. 

![](../images/base.png)

Ao passar o cursor sobre este quadrado, aparecem outros quatro quadrados ao lado, cada um representando um mapa base diferente. 

![](../images/base1.png)

O mapa base padrão no visualizador é o Argenmap, o mapa base do Instituto Geográfico Nacional (IGN). 

![](../images/base2.png)

Em seguida, está o mapa do OpenStreetMap. 

![](../images/base3.png)

Outro mapa base disponível é o Argenmap Cinza. 

![](../images/base4.png)

Também é possível selecionar o Argenmap Escuro como mapa base. 

![](../images/base5.png)

Por fim, está disponível o mapa de imagens de satélite da Esri.

![](../images/base6.png)
